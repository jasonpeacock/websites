HT16K33

So many LEDs, so much keyscan.

January 9, 2019

Introduction

A Rust driver for the Holtek HT16K33 “RAM Mapping 16*8 LED Controller Driver with keyscan”. The driver follows the embedded-hal implementation for Rust hardware abstraction.

Available on Github and crates.io as ht16k33.

The driver is part of the led-bargraph project, a Rust library & application for the Adafruit Bi-Color (Red/Green) 24-Bar Bargraph w/I2C Backpack Kit.
